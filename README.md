📚 JavaScript Projects Collection
A curated collection of beginner‑friendly JavaScript projects demonstrating core programming concepts, DOM manipulation, event handling, array operations, and basic UI interactions.
This repository serves as a hands‑on showcase of essential JavaScript skills through small, focused activities.

🚀 Overview
This repository includes multiple standalone mini‑projects, each designed to highlight a specific JavaScript concept or technique. Together, they form a solid foundation for understanding how JavaScript interacts with the browser and how to structure small applications.
The projects included are:
• 	📊 Mean Calculation Project
• 	🌗 Light & Dark Mode Toggle
• 	🏦 Banking System (OOP + Modules)
• 	🧮 Basic Calculator
• 	📦 Array Methods Mini Project
• 	📝 To‑Do List Application
Each project is self‑contained with its own HTML, CSS (when applicable), and JavaScript files.

• 	Mathematical operations
• 	Console output
