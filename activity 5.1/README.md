📊 Mean Calculation Project
A simple JavaScript experiment demonstrating how to compute the mean (average) of a dataset using basic programming constructs. This project is ideal for beginners learning JavaScript fundamentals such as arrays, loops, functions, and DOM‑independent scripting.

🚀 Overview
This project calculates the mean value of a predefined set of numerical data. It includes:
• 	A JavaScript function that computes the mean of an array
• 	A small dataset representing sample experimental values
• 	A console output showing the rounded mean
• 	A minimal HTML file that runs the script in the browser
Although simple, it demonstrates clean logic, modularity, and clarity — all essential for foundational programming.
